SLIDE 4: AI BASICS
------------------

Understanding AI Development

The "Magic": AI & App Dev Basics

The "Old Way" (Traditional Development)
Normally, to build a website, you'd have to write hundreds of lines of code in languages like HTML, CSS, and JavaScript. You'd also need to learn frameworks, databases, and how to set up a server. It's powerful, but has a very steep learning curve.

The "New Way" (AI-Powered Development)
Today, we're using a new approach. We will use AI to be our developer. We'll describe what we want in plain English (our "prompt"), and an AI platform like Firebase Studio will write all that complex code for us.

What We Are (and Aren't) Building
Our site today is a simple "static" site‚Äîit just displays information. More complex apps might need a database (to store user signups) or authentication (to let users log in). We aren't doing that today, but it's good to know those are the "next steps" in app development.
